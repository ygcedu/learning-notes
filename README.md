# iOfficeAI/AionUi

## 项目背景

AionUi 是一个免费、开源的 24/7 AI Cowork 桌面应用，支持 Gemini CLI、Claude Code、Codex、OpenCode 等多种 AI 代理，获得了 21.3k Stars 的极高关注度，是 AI Agent 客户端领域的明星项目。

## 核心技术/架构

项目采用 TypeScript (88.3%) 开发，辅以 Shell (6.2%)、CSS (3.0%)、JavaScript (1.3%)、Python (0.9%)。核心技术特点包括：

- **内置 Agent 引擎**：开箱即用，无需安装额外 CLI 工具
- **多 Agent 支持**：支持 12+ 种 AI 代理，统一界面管理
- **MCP 统一管理**：MCP 工具配置一次，自动同步到所有 Agent
- **跨平台**：支持 macOS / Windows / Linux
- **远程访问**：WebUI + Telegram / Lark / DingTalk 集成
- **定时任务**：Cron 定时执行，支持 24/7 无人值守

## 亮点与创新

1. **零配置启动**：安装后立即可用，无需复杂设置
2. **多模型支持**：支持 20+ AI 平台，包括 Gemini、Claude、DeepSeek、Ollama 等
3. **Office 助手**：内置 PPT、Word、Excel 生成能力
4. **文件智能管理**：批量重命名、自动分类、文件合并
5. **实时预览**：支持 10+ 格式即时预览
6. **免费开源**：完全免费，Apache 2.0 许可证

## 调研结论

AionUi 是目前功能最全面的开源 AI Agent 桌面客户端之一。相比 Claude Cowork，它支持更多模型、跨平台、远程访问和定时任务。对于需要使用多种 AI 代理的用户，AionUi 是理想的统一工作台。
