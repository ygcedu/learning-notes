# 21st-dev/1code

## 项目背景

1code 是一个开源的编程智能体客户端，作为 Claude Code、Codex 等编程代理的编排层，提供可视化界面和 Git 隔离开发环境，获得了 5.4k Stars 的关注。

## 核心技术/架构

项目采用 TypeScript (98.6%) 开发。核心技术特点包括：

- **多智能体编排**：支持同时运行多个 AI 编程代理
- **可视化 UI**：类似 Cursor 的桌面界面
- **Git Worktree 隔离**：每个任务在独立 Worktree 中运行，互不干扰
- **后台运行**：支持后台持续运行 AI 代理
- **MCP 插件市场**：支持 MCP 插件扩展

## 亮点与创新

1. **统一界面**：一个界面管理多种 AI 编程代理
2. **Git 隔离**：安全隔离的任务执行环境
3. **后台任务**：支持长时间运行的 AI 任务
4. **开源免费**：Apache 2.0 许可证

## 调研结论

1code 是编程智能体的高效编排层，适合需要同时使用多种 AI 编程工具的开发者。其 Git Worktree 隔离设计可以有效避免任务之间的相互干扰，是构建可靠 AI 编程工作流的好帮手。
