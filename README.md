# shareAI-lab/learn-claude-code

## 项目背景

learn-claude-code 是一个手把手教你构建类似 Claude Code 的 Agent 框架的教学项目，涵盖 MCP、协议封装、上下文管理等核心逻辑，获得了惊人的 50.8k Stars，是 AI Agent 教学领域的标杆项目。

## 核心技术/架构

项目采用 TypeScript (70.3%) 开发，辅以 Python (27.6%)、Shell (1.3%)。核心技术特点包括：

- **Agent Harness**：从零构建 Agent 框架的核心原理
- **MCP 协议**：Model Context Protocol 的深度解析
- **循环机制**：AI 代理的循环执行逻辑
- **工具系统**：工具的定义、调用和管理
- **规划能力**：任务分解和规划执行
- **上下文控制**：对话上下文的管理和传递

## 亮点与创新

1. **从零到一**：完整展示如何从零构建 Agent 框架
2. **原理深入**：不仅教怎么用，更教为什么
3. **Bash 哲学**：核心理念是"Bash is all you need"
4. **社区活跃**：50.8k Stars 说明其极高的质量和受欢迎程度

## 调研结论

learn-claude-code 是学习 AI Agent 开发的必读项目。无论你是想深入理解 Claude Code 原理，还是想构建自己的 AI 编程代理，这个项目都能提供宝贵的参考。建议与 Deepractice/PromptX 结合学习，后者提供了结构化的 Prompt 工程框架。
