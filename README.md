# 9Router 项目调研报告

## 项目背景

9Router 是一个开源的 AI 编程工具路由项目，旨在帮助开发者更高效地使用各种 AI 编程助手（如 Claude Code、Cursor、Cline、Codex、OpenClaw 等），通过智能路由机制连接到 40+ AI 提供商和 100+ 模型。

该项目解决了以下核心痛点：订阅额度每月浪费、速率限制导致工作中断、昂贵 API 成本（$20-50/月/提供商）、手动切换提供商繁琐等问题。

项目当前拥有 2,200 Stars 和 560 Forks，采用 MIT 许可证开源。

## 核心技术架构

### 技术栈

项目基于现代 Web 技术栈构建：运行时采用 Node.js 20+，前端框架使用 Next.js 16 配合 React 19 和 Tailwind CSS 4，数据存储使用 LowDB（基于 JSON 文件），流式响应通过 Server-Sent Events（SSE）实现，认证机制结合了 OAuth 2.0（PKCE）、JWT 和 API Keys。

### 智能三层 Fallback 机制

这是 9Router 的核心创新功能。当用户的订阅额度用尽时，系统会自动切换到更便宜的提供商；当便宜提供商也超出预算时，最终会自动切换到免费提供商。这种「订阅 → 便宜 → 免费」的自动降级机制确保了开发者可以持续工作，不会因为额度耗尽而中断编码。

具体流程为：用户请求首先发送到 9Router 本地服务，然后系统检查第一层（订阅级，如 Claude Code、Codex）的配额；如果配额耗尽，自动切换到第二层（便宜级，如 GLM $0.6/1M、MiniMax $0.2/1M）；如果仍然超出预算，最终切换到第三层（免费级，如 iFlow、Qwen、Kiro）。

### 格式转换引擎

9Router 支持 OpenAI、Claude、Gemini 三种 API 格式之间的无缝转换。这意味着无论用户的 CLI 工具发送什么格式的请求，9Router 都能将其转换为目标提供商所需的原生格式，极大地简化了多工具多提供商的集成复杂度。

### 多账户轮询

系统支持为每个提供商配置多个账户，当一个账户的配额用尽时，自动轮询到下一个账户，实现负载均衡和冗余备份。

## 亮点与创新

### 1. 零成本方案可行

通过组合 Gemini CLI（每月 180K 免费额度）+ iFlow（无限免费）+ Qwen（无限免费），用户可以实现完全免费的 AI 编程体验，这对于预算有限的个人开发者极具吸引力。

### 2. 配额实时追踪

Dashboard 实时显示每个提供商的 Token 消耗量和配额重置倒计时，帮助用户最大化利用订阅价值。

### 3. 广泛的 CLI 工具支持

支持 Claude Code、OpenClaw、Codex、OpenCode、Cursor、Cline、Continue、RooCode、Copilot、Kilo Code 等主流 AI 编程工具，覆盖面极广。

### 4. 灵活的部署方式

支持本地部署、VPS/云端部署、Docker 容器化部署、Cloudflare Workers 边缘部署，满足不同场景需求。

### 5. 云端同步

配置可以跨设备同步，用户在家用电脑和办公电脑上的设置可以保持一致。

## 与类似项目对比

### OneAPI / NewAPI

OneAPI 是国内开发者最熟悉的 API 聚合工具，主要提供统一的 API 格式转换和密钥管理功能。9Router 在此基础上增加了智能 Fallback、配额追踪、多账户轮询等高级功能，更适合需要高可用性的生产环境。

### OpenRouter

OpenRouter 是一个商业化的模型聚合平台，提供类似的多模型统一访问能力。9Router 作为自托管开源方案，具有成本优势（完全免费）和数据隐私优势（本地运行），但需要用户自行维护。

### CLIProxyAPI

这是 9Router 的灵感来源，最初是 Go 语言实现。9Router 使用 JavaScript/TypeScript 重写，降低了二次开发的门槛。

## 适用场景

9Router 特别适合以下用户群体：已订阅 Claude Code 或 Codex Pro 但经常遇到额度耗尽的开发者、希望以最低成本使用 AI 编程工具的个人开发者、需要 24/7 不间断 AI 编程能力的专业团队、以及需要统一管理多个 AI 账户和企业订阅的团队。

## 总结

9Router 是一个功能完善、设计精良的 AI 路由工具，通过智能 Fallback 机制和实时配额追踪，有效解决了 AI 编程工具使用中的成本和可用性问题。项目开源免费，部署简单，对于希望最大化利用 AI 订阅额度或追求零成本方案的开发者来说，是一个值得关注和尝试的工具。
