# sligter/LandPPT

## 项目背景

LandPPT 是一个基于 LLM 的演示文稿生成平台，能够自动将文档内容转换为专业的 PPT 演示文稿。该项目获得了 2.8k Stars 的高度关注，是 AI 办公工具领域的热门项目。

## 核心技术/架构

项目采用多语言开发，包括 JavaScript (41.4%)、Python (32.8%)、HTML (25.6%)。平台支持多种 AI 模型，提供丰富的模板和样式选择，支持 Docker 部署。项目最新版本 v0.1.8 发布于 2026 年 1 月 17 日，采用 Apache License 2.0 开源许可证。

## 亮点与创新

1. **端到端自动生成**：用户只需提供文档内容，平台即可自动完成内容理解、结构提取、模板匹配和视觉设计
2. **多模型支持**：支持切换不同的 LLM 模型，根据场景选择最优的 AI 模型
3. **模板系统**：提供丰富的专业模板库，支持自定义模板映射
4. **Docker 部署**：支持容器化部署，易于集成到企业工作流

## 调研结论

LandPPT 是 AI 办公自动化的优秀实践，适合需要快速生成演示文稿的场景。与传统的 PPT 模板工具相比，LLM 的引入使得内容理解和结构化更加智能。建议关注项目的模板系统和多模型支持能力，这两方面是产品差异化的关键。
