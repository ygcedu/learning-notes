# marswaveai/TypeNo

## 项目背景

TypeNo 是一款免费、开源、隐私优先的 macOS 语音输入工具，基于 Swift 开发，支持本地离线语音识别，无需联网即可将语音转换为文字并粘贴到当前应用中。项目获得了 811 Stars 的关注，是 macOS 平台热门的开源语音输入方案。

## 核心技术/架构

项目采用 Swift (94.6%) 和 Shell (5.4%) 开发。核心技术特点包括：

- **本地离线 ASR**：基于 sherpa-onnx 引擎，实现完全本地的语音识别，无需联网
- **Swift 原生开发**：充分利用 macOS 系统能力，提供原生应用体验
- **双通道实时上屏**：采用双通道架构，实现低延迟的实时转写体验
- **隐私优先**：所有语音数据在本地处理，不会上传到云端

## 亮点与创新

1. **离线识别**：完全本地运行，保护用户隐私，无需担心数据泄露
2. **极低延迟**：本地处理带来极低的响应延迟，输入体验流畅
3. **全局可用**：可以在任何应用中唤起使用，支持全局热键
4. **免费开源**：完全免费，源代码开放，社区驱动开发

## 调研结论

TypeNo 是 macOS 平台优秀的本地离线语音输入解决方案。与 Windows 的 doubao-ime-win 项目形成对比：一个基于本地离线 ASR，一个基于云端 API。对于注重隐私、需要离线工作环境的用户，TypeNo 是理想选择。
