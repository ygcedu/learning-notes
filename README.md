# Deepractice/PromptX

## 项目背景

PromptX 是领先的 AI 智能体上下文平台，旨在为 AI 开发者提供结构化的 Prompt 工程框架。该项目获得了 3.6k Stars 的高度关注，是 AI Agent 框架领域的重要项目。

## 核心技术/架构

项目采用 TypeScript (54.5%) 和 JavaScript (43.1%) 开发。核心技术特点包括：

- **结构化角色抽象**：支持定义具有特定行为的 AI 角色，实现角色化管理
- **记忆抽象**：管理对话上下文和长期记忆，支持跨会话记忆持久化
- **行动抽象**：定义 AI 可执行的操作序列，支持复杂任务编排
- **DPML 协议**：提供标准化的 Prompt 描述格式，便于 Prompt 的存储和交换
- **MCP 集成**：与 Model Context Protocol 无缝集成，可对接现有 AI 工具生态

## 亮点与创新

1. **软件工程思维**：将代码开发的结构化思想引入 Prompt 工程领域
2. **标准协议**：DPML 协议提供了 Prompt 的标准化描述方式
3. **企业级能力**：支持复杂场景下的多角色协作、长对话管理
4. **生态兼容**：通过 MCP 集成确保与现有 AI 工具的兼容性

## 调研结论

PromptX 是构建复杂 AI Agent 的优秀框架，特别适合需要多角色协作、长对话管理、复杂任务编排的场景。建议与 shareAI-lab/learn-claude-code 结合学习，可以深入理解 AI Agent 的架构设计原理。
