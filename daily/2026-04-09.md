## 技术总结

- **claude-code**: 基于 Bun 的 Claude Code CLI 逆向工程，TypeScript 全修复，适合学习 AI Agent 工具开发
- **PromptX**: 结构化 Prompt 工程框架，支持角色/记忆/行动抽象，DPML 协议 + MCP 集成
- **LandPPT**: LLM 驱动的 PPT 自动生成平台，支持多模型和模板映射
- **TypeNo**: macOS 本地离线语音输入法，Swift + sherpa-onnx，隐私优先
- **doubao-ime-win**: Windows 云端语音输入法，基于豆包 API，Rust 开发
- **mcp-server-duckdb**: DuckDB 的 MCP 服务实现，Python 开发，支持只读模式
- **aivectormemory**: 基于 MCP 的向量记忆服务，解决 AI 跨会话健忘问题
